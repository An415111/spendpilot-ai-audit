# SpendPilot

SpendPilot is an AI spend auditing platform designed for startups and engineering teams. It helps users analyze their AI tool subscriptions, identify unnecessary spending, and discover cheaper alternatives with actionable recommendations and projected savings.

## Features

- AI tool spend audit
- Savings recommendations
- Personalized AI-generated summaries
- Shareable public audit links
- Lead capture with email storage
- Responsive modern UI
- Persistent form state using localStorage

## Supported Tools

- Claude

- Windsurf
